# Feature: Veranstaltungen erstellen

## Beschreibung
Buchclub-Administrator kann Veranstaltungen für Buchclub-Treffen erstellen.

## Anforderungen
- Titel der Veranstaltung
- Datum und Uhrzeit
- Beschreibung

## Validierung
- Titel darf nicht leer sein.
- Datum muss in der Zukunft liegen.
- Veranstaltung muss gespeichert werden können.

## Akzeptanzkriterien
- Administrator kann Veranstaltung anlegen.
- Veranstaltung erscheint im Kalender.
- Mitglieder können die Veranstaltung sehen.
