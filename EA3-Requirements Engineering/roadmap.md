# Roadmap

## Aktueller Stand

**Phase: Idee / Konzept** — Noch kein Code, Constitution wird gerade erarbeitet.

**Team:** Solo-Entwickler (1 Person)

**MVP-Zieldatum: 1. August 2026**

---

## Phase 1 — MVP (bis August 2026)

Ziel: Die kleinstmögliche Version, die einem echten Buchclub echten Mehrwert bietet.

### Admin-Funktionen
- [ ] Buchclub anlegen und verwalten
- [ ] Mitglieder einladen und Rollen vergeben (Admin / Mitglied)
- [ ] Bücher zum Club-Katalog hinzufügen
- [ ] Veranstaltungen planen (Datum, Ort, Buch)

### Mitglieder-Funktionen
- [ ] Buch als gelesen markieren
- [ ] Buch bewerten (1–5 Sterne)
- [ ] Review / Kommentar zu einem Buch schreiben
- [ ] Teilnahme an einer Veranstaltung bestätigen / absagen

### Technische Grundlage
- [ ] Authentifizierung (Sign up / Login via Supabase Auth)
- [ ] Basis-Datenbankschema (Clubs, Mitglieder, Bücher, Events)
- [ ] Cross-platform Build (iOS + Android via React Native / Expo)

---

## Phase 2 — Wachstum (Q4 2026)

Ziel: Den Club mit Leben füllen und Engagement steigern.

- [ ] **Bewertungsstatistiken** — Durchschnittsbewertung pro Buch, live aktualisiert
- [ ] **Jahresrückblick** — Wie viele Bücher gelesen? Wer hat am meisten gelesen? Top-Bewertungen des Jahres
- [ ] **Push-Benachrichtigungen** — Neues Buch wurde hinzugefügt, Veranstaltung steht an, jemand hat ein Review geschrieben
- [ ] **Erinnerungen** — Automatische Reminder vor Veranstaltungen

---

## Phase 3 — Vision (2027+)

Ziel: BookCircle wird zur vollständigen Plattform für Buchclub-Kultur.

- [ ] **Bucharchiv** — Alle je gelesenen Bücher des Clubs auf einen Blick
- [ ] **Buchvorschläge** — Mitglieder können Bücher vorschlagen, Abstimmung im Club
- [ ] **Meeting-Hosting** — Integriertes Tool, um ein Treffen zu strukturieren (Agenda, Gesprächspunkte)
- [ ] **Diskussionsforen** — Pro Buch ein eigener Diskussionsbereich mit Threads
- [ ] **Offene Clubs** — Öffentliche Clubs, denen jeder beitreten kann

---

## Priorisierungsprinzip

> MVP first. Jede Funktion wird nur dann gebaut, wenn sie einem echten Buchclub echten Wert bringt — nicht weil sie technisch interessant ist.
